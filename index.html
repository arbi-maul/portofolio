Tentang Saya

Halo! Saya arbi. Portofolio ini disusun sebagai bagian dari tugas mata kuliah Cloud Computing, menampilkan eksplorasi dan implementasi saya dalam pengembangan aplikasi web sederhana.

Melalui proyek-proyek yang disajikan, saya bertujuan untuk mendemonstrasikan pemahaman dan keterampilan saya dalam pengembangan web dasar (HTML, CSS, JavaScript), penggunaan API untuk data real-time (aplikasi cuaca), dan implementasi logika bisnis sederhana (sistem kasir). Saya bersemangat untuk terus belajar dan berkontribusi dalam dunia pengembangan perangkat lunak.
